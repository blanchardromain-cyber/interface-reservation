Configuration EmailJS

Notifications automatiques aux professeurs lors d'une réservation

📘 Comment configurer EmailJS :
1. Créez un compte gratuit sur emailjs.com
2. Ajoutez un service e-mail (Gmail, SMTP…) → copiez le Service ID
3. Créez un template avec les variables {{to_email}}, {{to_name}}, {{resource}}, {{date}}, {{classe}}, {{notes}} → copiez le Template ID
4. Copiez votre Public Key depuis le compte EmailJS
5. Renseignez les champs ci-dessous et activez le toggle.

🔥 Configuration Firebase

Persistance partagée des réservations en temps réel

🔥 Comment configurer Firebase :
1. Allez sur console.firebase.google.com → Créer un projet
2. Ajoutez une app Web → copiez la configuration
3. Activez Firestore Database (mode test pour commencer)
4. Collez les valeurs ci-dessous et activez le toggle.
Une fois activé, toutes les réservations sont synchronisées en temps réel entre tous les navigateurs.
💡 Sans Firebase : les données restent dans le navigateur local (localStorage). Chaque utilisateur a sa propre copie.
Avec Firebase : un seul agenda partagé, mis à jour instantanément pour tous.

1. Créez un projet sur console.firebase.google.com
